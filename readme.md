# resistance

A game of deceit

## Setup

```sh
$ git clone https://github.com/azdavis/resistance.git
$ cd resistance/backend
$ go get
$ cd ../frontend
$ npm install
```

## Development

In one terminal:

```sh
$ cd backend
$ go build
$ ./resistance
```

In another terminal:

```sh
$ cd frontend
$ npm run start
```
